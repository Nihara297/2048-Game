# 2048-Game


A browser-based implementation of the classic **2048 puzzle game** built using **HTML, CSS, and JavaScript**. 


##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

- Use the **Arrow Keys** to move the tiles.
- When two tiles with the same number collide, they merge into one.
- After every valid move, a new tile (2 or 4) appears.
- Keep merging tiles to reach the **2048** tile.
- The game continues until no valid moves remain.
- Click **Restart Game** to begin a new game.

## ⚙️ Game Logic

- The game board consists of a **4 × 4 grid**.
- Two tiles are generated at the start.
- Each move shifts all tiles in the chosen direction.
- Adjacent tiles with equal values merge into one larger tile.
- The player's score increases based on the merged tile values.
- A new random tile is added after every successful move.
